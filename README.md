# bucket-brute
Aws S3 Tko Tool


```
usage: bucket-brute.py [-h] -u URL [-p PROXY] -b BUCKET

optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     No Such Bucket URL
  -p PROXY, --proxy PROXY
                        Proxy for debugging
  -b BUCKET, --bucket BUCKET
                        Bucket Name
```

[*] File Uploaded to S3 mytest [*]
